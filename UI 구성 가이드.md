# 똑똑용돈 UI 구성 가이드

> [API_SPEC.md](./API_SPEC.md) v2.1 계약과 확정 정책 기준으로, 화면별로 **구현해야 할 UI 요소**를 정리한다. 화면 코드(O/S/F/G)는 API_SPEC 부록 B와 동일.
> 상태 표기: 🆕 신규 화면 / ✏️ 기존 목업 수정 / 🗑️ 제거 / ✅ 유지(데이터 바인딩만)

## 0. 전 화면 공통 원칙

- **서버 값만 표시**: 포인트·잔액·진행률·날짜("오늘" 판정 포함)는 전부 API 응답 값. 하드코딩 값(84,500P·3/5·"7월 21일 화요일" 등) 전면 제거.
- **시니어 접근성**: 큰 글씨 모드(현행 `seniorTextScale` 0.9~1.35 유지), 큰 터치 타깃, 짧은 문장.
- **비진단 톤**: 케어·리포트·라벨 문구는 활동 관찰 표현만("기억력이 떨어졌습니다" 류 금지 — 미션게임 기획서 §5 피해야 할 표현 준수).
- **똑똑이 화자 원칙**: 시니어 앱의 안내·인사·보상 문구는 돼지 마스코트 '똑똑이'가 말하는 톤으로 통일한다("할무니, 오늘 하루 어떠셨어요?" 류 — 손자처럼 친근한 존댓말). guardian 앱은 중립 톤 유지.
- **음성 데이터 취급(전 화면)**: 시니어 **음성 일기**의 음성 원본은 기기 밖으로 내보내지 않는다(on-device STT — API_SPEC §1.9 특칙). 전사 원문은 **기본 비공개** — 시니어가 공유 설정을 켠 경우에만 guardian 화면에 노출 가능(그 외에는 리포트·대시보드의 요약 신호만). 공유 여부는 시니어 본인만 변경할 수 있다. 자녀 **음성 응원 편지**의 오디오는 전달 목적으로만 서버에 저장된다 — 재생은 서명 URL로, 무료 발신분은 24시간 후 사라짐을 발신·수신 양쪽 UI에 명시. 모든 녹음 화면은 녹음 중 대형 표시 + 언제든 중단·재녹음 + 전송 전 확인을 갖춘다.
- **프리미엄 게이트 UX**: 서버 402 수신 시 잠금 카드(블러 + 프리미엄 배지 + "구독하고 보기" CTA) → 페이월(F-13) 이동. 잠금 여부를 클라이언트가 자체 판정하지 않는다(서버 게이트 정본 — API_SPEC §11).
- **상대 시간**: "방금 전/2시간 전/어제"는 클라이언트가 `occurred_at`(ISO 8601)으로 렌더링.
- **오프라인 큐 UX**: 미전송 완료 건은 "적립 예정" 표시, 로컬 잔액 가감 금지. 409 수신 시 큐 제거.
- **에러 UX**: 401 토큰 갱신→재시도, 403 화면 차단, 429 백오프 안내, 5xx 재시도 (API_SPEC §1.3).
- **guardian 데이터 디스크 저장 금지**: 대시보드·추이·리포트·케어알림·달력은 메모리 캐시만.

## 1. 온보딩·인증

| 화면 | 상태 | 구현 요소 |
|---|---|---|
| O-1 스플래시 / O-2 인트로 캐러셀 | ✅ | 현행 유지 |
| O-4 로그인 | ✏️ | **Apple 로그인 버튼 + 카카오 로그인 버튼**(카카오톡 앱 전환). "휴대폰 번호로 계속하기" 버튼·`PhoneLoginFlow`(OTP 키패드 포함) 🗑️ 제거 |
| O-5b 약관 동의 | ✏️ | 5종 체크리스트(필수 4 + 마케팅 선택), 전체 동의, 민감정보 안내 — 동의 결과를 `POST /auth/session`의 `consents`로 전송(로컬 저장 제거). 각 항목 탭 시 전문 열람: `tos` → 이용약관.md, `privacy`·`sensitive_data` → 개인정보처리방침.md (스크롤 가능한 전문 뷰, 시니어 큰 글씨 적용) |
| 🆕 가입 정보 입력 | 🆕 | **실명·이메일·전화번호·성별·생년월일 입력 폼**(수집 항목 확정). Apple/카카오 프로필 값 프리필, Apple "Hide My Email" 시 실이메일 안내 문구. 유효성 검사(이메일 형식, 전화번호 형식, 생년월일 데이트피커) |
| O-3 역할 선택 | ✏️ | 자녀/부모님 카드 2택 — 결과를 session 요청 `role`로 전송(UserDefaults 저장 제거). **가입 후 변경 불가 안내 문구** |
| O-6 가족 연결(자녀) | ✏️ | 초대 코드 **4자** 표시(6자 하드코딩 "7Q9K2M" 수정), 코드 복사, 카카오톡 공유, QR(`qr_payload`), **연결 대기 상태(초대 상태 3~5초 폴링) → 연결 완료 전환**, 24시간 만료 카운트다운 |
| O-7 가족 연결(시니어) | ✏️ | 4칸 코드 입력(유지), 오류 상태 표시(잘못된 코드 404/만료 400/이미 구성원 409 각각 다른 안내), 레이트리밋 429 안내 |
| O-8 환영 | ✏️ | 연결 상대 실명 표시(서버 `members` 응답 기준, Mock 이름 제거) |

## 2. 시니어 앱

| 화면 | 상태 | 구현 요소 |
|---|---|---|
| S-1 홈 | ✏️ | **똑똑이 마스코트 + 인사말**(`display_name` 호명 발화, 탭하면 S-20 음성 일기 진입 — "오늘 하루 어땠는지 똑똑이에게 들려주세요" 부제, `voice_diary.enabled=false`면 동의 안내 시트), **우체통**(`mailbox` — `unread_count > 0`이면 귀엽게 흔들리는 애니메이션 + 배지, 탭 → S-21 편지함), **오늘의 미션 카드**(추천 rank 1: 제목·이모지·예상 시간·기본 보상 + **인지 라벨 칩**("기억력") + **보너스 배지**("+보너스 20P")), 오늘 참여 링(`missions_completed`/`missions_total`=5), 체크인 배너(+10P, `checked_in` 상태), 다른 활동 리스트, 잔액. **걸음 카드 🗑️ 제거** |
| S-2 체크인 | ✏️ | **똑똑이 대화형 리스킨**: 똑똑이가 묻고("오늘 기분은 어떠세요?") 시니어가 선택지로 답하는 말풍선 UI — 기분 5단계 + 수면 품질 3단계 2스텝, 데이터 구조는 기존 그대로(**선택값을 코드(mood 1~5, sleep_quality 1~3)로 전송**). 완료 시 서버 응답의 `earned_points`(+10P, 최초 1회만)를 똑똑이 발화로 표시 |
| 🆕 S-20 똑똑이 음성 일기 | 🆕 | 부제 **"오늘 하루 어땠는지 똑똑이에게 들려주세요"**. 첫 진입 시 **`voice_diary` 동의 화면**(무엇을 저장하고 무엇을 저장하지 않는지 큰 글씨 설명 — "목소리는 저장되지 않아요, 이야기 내용만 기록돼요", 기본 가족 비공개 안내). **"가족에게 내 이야기 보여주기" 토글**(기본 꺼짐, 시니어 본인만 — 켜기 전 "자녀분이 이야기 전체를 읽을 수 있어요" 확인, 언제든 끄기 가능. 마이페이지 설정에도 동일 토글). 똑똑이를 길게/한 번 눌러 녹음 시작 — **녹음 중 대형 표시 + 언제든 중단**, 기기 내 STT 전사 미리보기 → 시니어가 확인 후 "들려주기" 버튼으로 전송(무단 자동 전송 금지). 오늘 이미 들려줬으면 "다시 들려주기"(덮어쓰기 안내). 지난 일기 목록·삭제(본인만). 마이크·음성인식 권한 거부 시 안내 화면 |
| 🆕 S-21 우체통·편지함 | 🆕 | 홈 우체통 탭으로 진입. 새 편지 열기: **편지지가 펼쳐지는 연출** → STT 전사 편지 본문(큰 글씨) + **음성 재생 버튼**(자녀 목소리), 발신자 표시(실명 또는 "익명의 응원"). 열람 시 `:listened` 호출(우체통 흔들림 해제). 편지 목록(지난 편지 계속 접근) — **무료 발신분은 24시간 후 사라짐** 안내("편지가 하루 뒤 하늘로 돌아가요" 톤), 영구 보관 편지는 보관 배지. 만료 임박 카운트다운 |
| S-3 미션 목록 | ✏️ | 세그먼트: 오늘의 미션(배정 5개) / 인지 게임(시드 6종 카탈로그) / 교육. 카드 요소: 이모지·제목·예상 시간·기본 보상·**인지 라벨 칩**·**보너스 배지**·완료 체크. 교육 탭의 `premium` 콘텐츠는 프리미엄 배지 표시. **걷기 항목 🗑️ 제거** |
| S-4 게임 인트로 | ✏️ | 게임 규칙·보상(기본+보너스 합산 표시)·인지 라벨 — 시드 6종 공용 |
| S-5 게임 플레이 | ✏️🆕 | **신규 3종 화면 구현**: 숫자·기호 짝맞추기(규칙표+선택지), 다른 그림 찾기(두 그림 비교 터치), 신호에 맞춰 누르기(go/no-go 버튼). **목업 2종 완성**: 단어·그림 분류(범주 상자 드래그/탭), 일상 순서 맞추기(카드 배열). 순서 기억하기 ✅ 유지. **전 게임 공통: 문항별 반응시간 기록 + 힌트 버튼 실동작(사용 횟수 기록)**. 같은 그림 찾기·소리 퀴즈·길 찾기 화면 🗑️ 제거 |
| S-6 게임 결과 | ✏️ | 성공/실패는 **서버 `complete` 응답 기준**(클라 판정 제거), 실패 시 재도전 버튼(assignment 유지), 정답 요약 문구 |
| S-7 보상 획득 | ✏️ | `earned_points`(기본+보너스 분리 표시 "+40P +보너스 20P"), 오늘 진행(`today.missions_completed/total`) — 하드코딩 3/5 제거 |
| S-8 용돈함 | ✏️ | 잔액(`balance`)·원화 환산(1P=1원), 적립 내역 리스트(서버 `emoji`·`title`·`delta`·상대 시간), 출금·상점 진입 버튼 |
| S-9 걸음수 | 🗑️ | **화면 전체 제거**(권한·상세 둘 다) — 기능 폐기 확정 |
| S-10 마이페이지 | ✏️ | 실명·연결 가족(서버 `members` 기준), 글자 크기 조절 유지, 알림함·알림 설정·로그아웃 |
| S-12 출금 | ✏️ | 출금 가능 잔액(서버), 금액 선택 칩, 계좌 표시(데모 — 마스킹), 수수료 500원 안내, 신청 결과(`requested`) 상태 |
| S-13 상점 | ✏️ | 보유 포인트(서버), **카테고리 섹션 그리드**(`category` 기준 — 꾸미기/테마/기프티콘/교육/건강/제휴, `price_points` 숫자, 무료=0), 제휴(`partner`) 상품은 제휴 표기 배지, **구매 버튼 실동작**(잔액 부족 400 안내, 구매 완료 후 잔액 갱신, `fulfillment.status=demo` 상태 안내), **"가족에게 선물" 버튼**(상품 상세 — 수령인 선택(가족 구성원) → 미션으로 모은 용돈으로 자녀·손주에게 선물, 완료 시 "선물을 보냈어요" 연출), 선물 내역 진입 |
| S-15 교육 상세 | ✏️ | 본문을 서버 `body_md`(마크다운 렌더) 또는 `video_url`로 표시(하드코딩 본문 제거), 완료 버튼 → `earned_points` 표시(재완료 0P). **`premium` 콘텐츠**: 프리미엄 배지 + 무료 티어(가족 구독 없음)면 잠금 안내(402 — "자녀분 구독으로 열려요" 톤, 시니어에게 직접 결제 유도 금지) |
| S-17 주간 목표 | ✏️ | 목표 진행바(`earned/target`), 보상 조건 리스트 — **연속 참여·주간 목표 2조건**(걸음 5일 조건 🗑️ 제거), 각 조건 달성 상태·진행도 |
| S-18 알림함 | ✏️ | 알림 리스트(타입 아이콘·제목·상대 시간·**읽음 상태**), 무한 스크롤(커서), 읽음 처리. **푸시 인프라 공통**: 로그인 후 알림 권한 요청 → APNs 토큰 `POST /me/devices` 자동 등록, 로그아웃 시 해제, 푸시 탭 시 해당 화면 딥링크(케어 푸시 → F-1/F-5) |
| 🆕 S-19 활동 달력 | 🆕 | **월간 그리드**: 일별 스탬프 3단계(`complete`/`partial`/`none`), 월 이동, 요약(활동일·올클리어·최장 스트릭). **셀 탭 → 일별 상세**(그날 미션 5개 상태·점수, 체크인 기분/수면, 획득 포인트) |

## 3. 가족(자녀) 앱

| 화면 | 상태 | 구현 요소 |
|---|---|---|
| F-1 대시보드 | ✏️ | 부모 선택 칩(시니어 여러 명 — `members` 기준), **케어 상태 카드**(`safe/caution/alert` 색·마스코트·메시지), 주간 참여율+증감, 오늘 미션(x/5), **최근 활동 시각**(`last_active_at`), 최근 활동 리스트, **음성 응원 보내기 버튼 → 녹음 시트**(똑똑이 연출 + 60초 녹음 + on-device STT 전사 미리보기 + **익명/실명 토글** → multipart `POST /cheers`, 일 3회 초과 429 안내. **전사 미리보기 단계에서 로컬 금칙어 검사** — 감지 시 발송 버튼 비활성 + 재녹음 안내, 서버 400(부적절 표현) 수신 시 사유 카테고리 안내. **"편지는 24시간 뒤 사라져요 — 프리미엄은 영구 보관" 업셀 라인**), 가족 선물 내역(`GET /gifts`) 진입, **"부모님 이야기" 진입**(부모님이 공유를 켠 경우에만 노출 — 음성 일기 원문 열람, 읽기 전용) |
| F-2 활동 추이 | ✏️ | 지표 탭 3종(정답률/반응시간/참여빈도) — **탭별로 별도 시계열 호출**(현재 공용 mock 1세트 수정), 7일/30일 라인 차트(`series`), 요약 수치(`current`·`delta`·평균 반응시간·주간 참여). **30일 탭 = 프리미엄**(무료는 7일만, 잠금 카드 → F-13) |

## 5. 구현 우선순위 제안

1. **P0 (계약 성립)**: 로그인(Apple+카카오)·가입 정보 입력·약관·역할 → 가족 연결(4자 코드+폴링) → 홈·미션 목록·순서 기억하기(계측 추가)·결과/보상 — 코어 루프 1개 게임으로 관통
2. **P1 (데모 완성)**: 체크인, 용돈함·원장, 대시보드·추이·리포트, 주간 목표, 걸음/폐기 게임 제거 정리. **병행 가능(리텐션 접점)**: 똑똑이 리스킨(S-1 인사말·S-2 대화형 — UI 작업만, 서버 무관)
3. **P2 (확장)**: 신규 게임 3종 + 목업 2종 완성, 활동 달력(S-19/F-12), 상점·출금·충전(카테고리 섹션·충전 내역·**가족 선물하기** 포함), 알림함·알림 설정, 케어 알림·패턴·히스토리, **똑똑이 음성 일기(S-20)** — 동의 UX·on-device STT·전사 암호화·배치 분석까지 한 묶음(부분 출시 금지: A-7 ⑦ 보안 전제 미충족 상태로 녹음 기능만 먼저 여는 것 금지), **음성 응원 편지(F-1 녹음 시트 + S-1 우체통 + S-21 편지함)** — 오브젝트 스토리지·24h 파기 배치 확정 후(A-7 ⑨)
4. **P3 (수익화 검증 — P2 완료 후)**: 프리미엄 구독 게이트(백엔드 entitlement + 402) + 페이월(F-13) + 잠금 UI(F-2/F-3/F-5/F-9/F-12) — 데모 구독으로 게이트 플로우 검증, 실결제 전환은 API_SPEC 부록 A-7 ②·④ 결정 후
