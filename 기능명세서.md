### 기능 명세서

**기능 1. 시니어 참여 홈**

- 목적: 시니어가 매일 앱에 쉽게 진입해 당일 미션을 확인하고 참여하도록 유도
- 주요 기능: 오늘의 미션 확인, 미션별 보상 포인트 확인, 오늘의 참여 현황 확인, 미션 시작, 완료 상태 확인
- 핵심 가치: 용돈을 트리거로 “오늘 무엇을 하면 되는지” 직관적으로 안내하고 자연스러운 반복 참여를 유도
- 데이터 연결: 미션 시작·완료 여부, 참여 시각, 완료율, 획득 포인트를 저장

**기능 2. 미션형 참여 콘텐츠**

- 목적: 시니어의 부담 없는 활동 참여를 유도하고 반복 수행 데이터를 축적
- 주요 기능: 인지 활동 게임, 교육형 콘텐츠, 일상 체크인, 생활 습관 미션 등 당일 미션 수행
- 게임 데이터: 정답률, 반응 시간, 오류 횟수, 재시도 여부, 힌트 사용 여부, 참여 빈도
- 핵심 가치: 병원식 검사보다 부담이 적은 활동을 통해 시니어의 참여를 만들고, 개인별 활동 변화와 참여 패턴을 관찰할 수 있는 데이터 생성
- 데이터 활용: 축적된 데이터는 자녀 대시보드의 주간 요약과 다음 미션 추천에 활용
- 미션 게임 예시: 인지 활동 게임
당일 미션은 시니어가 짧고 부담 없이 참여할 수 있는 게임 형태로 제공하며, 수행 결과는 자녀 대시보드의 활동 변화 요약에 활용한다.

| 예시 게임 | 부담 없는 참여 | 데이터 구조화 | 대시보드 연결 |
| --- | --- | --- | --- |
| 숫자·기호 짝맞추기 | 높음 | 높음 | 반응 속도·정답률 변화 |
| 소리 듣고 그림 고르기 | 중간~높음 | 높음 | 선호 콘텐츠·응답 패턴 |
| 오늘 할 일 기억하기 | 높음 | 높음 | 생활형 참여·누락 변화 |
| 장보기 목록 기억하기 | 높음 | 높음 | 기억 활동·생활 주제 선호 |
| 신호에 맞춰 누르기 | 매우 높음 | 매우 높음 | 반응 시간·집중 흐름 |

게임 데이터는 의료적 진단이 아닌 개인별 참여 패턴과 활동 변화 관찰을 위한 지표로 활용한다.

**기능 3. 용돈 기반 보상 구조**

- 목적: 미션 참여를 지속시키는 동기를 제공하고 활동 결과를 보상 흐름과 연결
- 주요 기능: 미션별 포인트 적립, 누적 포인트(용돈) 확인, 주간 목표 포인트, 용돈 보상 조건 및 달성 상태 표시
- 핵심 가치: 용돈을 단순 송금이 아니라 시니어의 지속적인 참여를 유도하는 행동 트리거로 활용
- MVP 범위: 실제 결제나 송금 없이 포인트와 보상 예정·달성 상태를 데모로 제공
- 데이터 연결: 미션 완료 기록과 포인트 변화를 가족 대시보드 및 AI 리포트에 반영

**기능 4. 가족 대시보드**

- 목적: 자녀가 부모님의 참여 현황과 최근 활동 변화 AI를활용하여 대시보드로 한눈에 이해하도록 지원
- 주요 기능: 오늘의 참여 여부, 주간 미션 완료율, 활동별 참여 기록, 포인트 변화, 참여 패턴 요약, 관리 포인트 확인, **케어 알림·푸시**(부모님 활동이 3일 이상 없거나 이상 신호 감지 시 자녀에게 푸시 알림 — 앱을 열지 않아도 도달, 구독 티어 무관)
- 핵심 가치: 원시 활동 기록을 단순히 나열하지 않고, 자녀가 부모님의 최근 상태와 변화 흐름을 빠르게 이해할 수 있는 가족 케어 인사이트 제공
- 데이터 표현: 진단이나 위험 판정이 아닌 참여 증가·감소, 선호 활동, 수행 변화 중심으로 제공

**기능 5. 개인화 AI 분석 및 추천**

- 목적: 누적된 활동 데이터를 요약하고 시니어별 맞춤형 미션 추천으로 연결
- 주요 기능: 활동 데이터 구조화, 개인별 변화 패턴 분석, 가족용 주간 요약 생성, 다음 미션 추천, 관리 포인트 제공
- 핵심 가치: 단순 기록에 머무르는 활동 데이터를 가족이 이해할 수 있는 관리 정보와 개인화된 참여 경험으로 전환
- 안전 기준: AI는 질병·치매 여부를 진단하기보단, 활동 참여와 수행, 인지 상태 변화 체크 분석 및 요약
- 운영 구조: 상세 설계 필요

**기능 6. 가족 소통 (음성 응원 편지·형제자매 연결)**

- 목적: 용돈·데이터 중심의 가족 연결을 정서적 소통으로 확장하고, 형제자매 유입으로 가족 단위 리텐션을 강화
- 주요 기능: 자녀가 똑똑이를 통해 **목소리로** 응원 편지를 보냄(60초 녹음 + 기기 내 STT 전사, 익명/실명 선택) → 부모님 홈의 **우체통이 귀엽게 흔들리며** 도착 알림 → 열면 **편지지 UI**에 전사 편지가 펼쳐지고 음성이 재생됨, 편지함에서 계속 접근. 기존 초대 플로우를 재사용한 형제자매(guardian) 추가 연결, 가족 지갑 충전 기여 내역(충전자별) 표시
- 보존 정책: **24시간 휘발 또는 영구 보관 둘 중 하나** — 발송 시점의 발신자 멤버십으로 확정(무료 발신 = 24시간 후 자동 파기·사후 전환 없음, 프리미엄 발신 = 영구 보관·구독 해지 후에도 유지) — 구독 업셀 접점
- 핵심 가치: 부모님에게는 자녀 목소리로 전해지는 응원의 체감을, 자녀에게는 송금 이상의 정서적 소통 접점을 제공
- 안전 기준: 응원 오디오·전사는 전달 목적으로만 암호화 저장하고 프로파일링 분석에 투입하지 않음, 접근은 수신 부모님·발신자만, 익명은 표시상 익명(서버는 발신 기록 유지). **발송 시 안전 필터**: 욕설·모욕·위협·위험 표현·금전 요구 의심 문구를 자동 판별(서버 금칙어·패턴 검사 + 발송 전 앱 내 1차 검사)해 감지 시 발송 차단
- MVP 범위: 분담결제·N분의1 정산은 실결제 확정 전 범위 외 — 충전 기여 표시까지만

**기능 7. AI 정서·생활 동반자 '똑똑이' — 음성 일기**

- 목적: 돼지 마스코트 '똑똑이'를 시니어 경험의 화자로 세우고, "오늘 하루 어땠는지 똑똑이에게 들려주세요"라는 음성 대화로 일상·기분·활동 이야기를 꾸준히 기록
- 주요 기능: 홈에서 똑똑이를 눌러 음성으로 하루 이야기 → 기기 내 STT 전사 → 시니어 확인 후 전송(하루 1회), 지난 일기 열람·삭제(본인 전용), 홈 인사말·체크인의 똑똑이 대화형 UI
- 분석 데이터: 이야기 내용(정서·주제 요약)과 발화 특징 — 발화 속도·멈춤, 같은 말 반복, 이야기 개연성, 단어를 떠올리지 못하는 신호(필러·어휘 다양성) 등의 **변화 추이**
- 핵심 가치: 시니어에게는 손자에게 이야기하듯 부담 없는 기록 경험을, 가족에게는 참여·정서 변화의 참고 신호와 소통 포인트를 제공
- 안전 기준(확정 — API_SPEC §1.9 특칙): **음성 원본 서버 미저장**(on-device STT, 오디오 업로드 API 없음) · 전사 텍스트는 **별도 명시 동의(opt-in) + 암호화 저장 + 보존 기간 후 원문 파기** · **전사 원문은 기본 비공개 — 가족 공개 여부는 부모 사용자 본인이 설정으로 결정**(켜면 가족 열람 가능·감사 로그, 언제든 끄기 가능, 꺼진 상태에서는 요약 신호만 공유) · 분석 결과는 의료적 진단이 아닌 참여·정서 변화 참고 신호로만 표현(비진단 원칙) · 동의 철회 시 전체 삭제
- 데이터 활용: 축적된 발화 특징·정서 신호는 가족 주간 리포트·참여 패턴의 소통 포인트로 연결, 향후 캐릭터 구독·디지털 선물·굿즈 확장의 기반

**기능 8. 프리미엄 구독 (자녀 대상 심화 인사이트)**

- 목적: payer(자녀)·user(부모)가 분리된 구조에서 가장 지불 의사가 명확한 자녀에게 반복과금 모델을 검증
- 주요 기능: 3티어(free/premium/premium_ai) 구독, 무료(오늘 참여·케어 상태·최신 리포트 1건·7일 추이) / 프리미엄(30일 추이·리포트 아카이브·수동 생성·참여 패턴·부모님 활동 달력·케어알림 전체 히스토리·프리미엄 교육 콘텐츠 가족 상속), 페이월·잠금 UI
- 핵심 가치: 이미 만든 대시보드·리포트 위에 결제 게이트만 얹는 구조 — 신규 화면은 페이월 1개
- 안전 기준: 게이트는 서버가 강제(402), 시니어 본인의 자기 데이터 열람은 티어와 무관하게 항상 허용, 시니어에게 직접 결제 유도 금지
- MVP 범위: 데모 구독 상태로 게이트 플로우 검증 — 실결제·가격은 결정 대기(API_SPEC 부록 A-7 ②·④)

**기능 9. 포인트 상점 커머스 확장 · 가족 선물하기**

- 목적: 용돈 포인트 소비처를 넓혀 포인트 순환을 강화하고, 가족 간 선물 순환으로 리테일 마진·제휴(B2B) 수익의 기반을 마련
- 주요 기능: 상점 카테고리 확장(꾸미기·테마·기프티콘·교육·건강·제휴), 건기식·보청기 등 시니어 특화 상품 카테고리, 제휴 상품(`partner`) 지면 — 별도 광고 시스템 없이 상점 상품으로 흡수. **가족 선물하기**: 포인트로 현물 상품(기프티콘·건강 상품 등)을 구매해 가족 구성원에게 선물 — 시니어는 미션으로 모은 용돈으로 자녀·손주에게, 자녀는 가족 지갑 충전 잔액으로 부모님께. 선물 도착 알림 + 가족 선물 내역
- 핵심 가치: "부모님이 모은 용돈으로 손주에게 커피 한 잔" — 보상 포인트가 가족 정서 순환으로 이어지는 가족 커머스. 제휴 상품이 선물로 오가며 B2B 채널(시니어 용품·건기식 제휴)의 자연스러운 접점이 됨
- 안전 기준: 실결제 전환 시 자금 이동 채널 악용(세탁·보이스피싱) 방지를 위해 충전·현금성 선물에 한도, 신규 가족 연결 후 냉각 기간, 이상 거래 탐지를 적용 (정본: 보안 문서.md §4.6)
- MVP 범위: 실발급·실물 배송 없이 전 상품 데모 이행 상태(배송 주소 미수집) — 실결제 범위 확정 후 전환
